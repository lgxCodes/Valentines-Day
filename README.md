# Will You Be My Valentine?

A fun, interactive Valentine's Day webpage that asks the big question — with a twist. The "No" button dodges away every time you try to click it, leaving "Yes" as the only real option.

## Live Demo

[Live Demo](https://lgxcodes.github.io/Valentines-Day/)

## How It Works

- Click **Yes** and a fullscreen celebration appears with floating hearts
- Try to click **No** and watch it run away to a random spot on the screen
- Works on both desktop (mouse) and mobile (touch)

## Features

- Dodging "No" button that escapes your cursor/tap
- Fullscreen celebration screen with animated floating hearts
- Responsive design for mobile, tablet, and desktop
- Smooth fade-in animations
